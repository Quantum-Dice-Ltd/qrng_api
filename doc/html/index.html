Overview

This Manual provides detailed descriptions of the programming interface of the Quantum Dice QRNG library.

The QRNG API provide an interface between a user application and Quantum dice's QRNG hardware, with these API the user is able to fetch truly random Quantum Random Numbers from the hardware.

Normally, the QRNG APIs need Quantum Dice's QRNG hardware to perform their operation, without this hardware the functions would return an error, however the APIs have a fail-over mechanism that enables them to return PRNs (Pseudo-Random Numbers) in a case where the hardware is not available, but the caveat here is that PRNs are not as safe as QRNs (Quantum Random Numbers). It is not encouraged to use the APIs without the hardware, this mechanism is only provided for testing and in rare cases where there could have been physical damage or tampering on the hardware.
